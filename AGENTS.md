# Repository Guide

This repository is the static Prospex investor narrative deployed with GitHub Pages.

## Map

- `index.html`: investor story, content hierarchy, accessible page structure.
- `styles.css`: brand tokens, responsive layout, motion, and visual system.
- `script.js`: workflow/query interactions, print behavior, and scroll reveals.
- `assets/`: source and generated visual assets plus local open-source fonts.
- `memory.md`: durable product beliefs, deployment facts, and recurring pitfalls.
- `design-qa.md`: blocking visual-comparison report for the latest redesign.

Read `memory.md` before changing positioning, brand treatment, or deployment behavior. Update `design-qa.md` whenever a visual target is implemented or materially revised.
